# Descripción
Esta API expone un modelo de regresión basado en el dataset 'real_state_ecuador_dataset'  
Permite predecir el precio de alquiler de una vivienda en función de sus características.

# Justificación del modelo elegido
Se utilizó un modelo de regresión lineal porque es un método sencillo, interpretable y adecuado como punto de partida para analizar la relación entre las características de las viviendas y su precio de alquiler. Este modelo permite identificar el impacto de cada variable sobre el precio y sirve como referencia inicial antes de aplicar técnicas más avanzadas.

El desempeño del modelo evaluó las métricas MAE, RMSE y R². El MAE y el RMSE muestran el error promedio y típico de las predicciones, mientras que el R² indica el grado en que las variables explican la variación en los precios. Estos resultados permiten valorar la precisión del modelo y entender sus limitaciones.
